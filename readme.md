## Make sure you install

- [on-my-zsh](https://ohmyz.sh/)

## Install

```bash
git clone https://github.com/OtterArkar/otter-zsh ~/.oh-my-zsh/custom/themes

```

then edit your _.zshrc_ file:

```bash
ZSH_THEME="otter-minimal"

or

ZSH_THEME="otter-cool"
```

```bash
source ~/.zshrc
```
